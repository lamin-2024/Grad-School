How to use terminals (cmd equivalent in windows) on Unix

First check which environment you in
		echo $SHELL

To check the environment you are in 
	env 

To check the group use either commands
	groups
	groups username
	
Check the list of directory in your current shell
			ls -a
			
How to check for the available modules in your current environment
		module avail
		
How to load a module once yor know which module you want
		module load "name of module without quotes"
		
Verify the loaded module: to check which module is currently running in your session
		module list
		
Unload a Module
		module unload "name of module without quotes"
		
Unload all modules or purge all modules
		module purge

note the following commands		
module avail          # List available modules
module load gcc/9.3   # Load a specific module (e.g., GCC version 9.3)
module list           # List loaded modules
module unload gcc/9.3 # Unload the module
module purge          # Unload all modules
