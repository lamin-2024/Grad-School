Install SystemC
	Open a new terminal
	check yor Downloads folder using the command
		to get to the Download folder:   							 ~/Downloads/
		to get the list of contents in the Downloads folder in ltr form or long time reverse (ltr) form:		  ls -ltr
		to know the function of a comman use man (command): man ls
		to make a directory for systemC: mkdir ~/systemC
		to go to the directory: cd ~/systemC
			cd = change directory 
			~  = takes you to the home directory
		Note install systemC as a module
		to unzip the download file for systemC: gzip -d -c ~/Downloads/systemC-3.0.0.tar.gz | tar xvf
			-d = decompress
			-c = standard out
****************************to extract files from archive tarred and comppressed usinf gzip.******************
use any of the following commands 
gzip -d -c <archived filename> | tar xvf -
tar xvfz <archived filename>

note:	tar = archive
		x   = extract
		v   = to visualize the process
		f   = file

		to check to see if you have the extracted file: ls
		to get to the home directory: cat /home 
		while in home directory
			to get to the file you extrated use the following commands
				cd ~/systemC/
				then: cd ~/systemC-3.0.0/
				list the content: ls
				
			to get to the configuration: ./configure --help
			to check the available modules: modules available
			
			to add a module: module add module name
			in our case: module add compiler/gcc/v7.3.0
			
			to check if the module is added: modules list
			currently loaded Modules: will show
			
			to check the version of the module: gcc --version
			to delete a module: module delete module name
